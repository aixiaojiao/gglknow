如果您通过Chrome Web Store安装，插件将自动更新到新版本。

## 🎯 使用提示

### 解决设置问题
如果您之前遇到"请先设置保存路径"的问题：
1. 点击插件图标打开设置
2. 重新设置保存路径（如：`TwitterCollections`）
3. 点击"保存设置"确认成功
4. 使用"🔧 调试工具"验证设置有效性
